## Remaining / New Findings

### MUST-FIX (blocks `--out` workflow GO)

**CRITICAL — `assertSafeOutDir` does not check `.assetsignore`; PII skeletons can reach deployable asset paths**
`scripts/printer-intake-scout.js` — `assertSafeOutDir` function

- **Mechanism:** The guard now calls `git check-ignore` to verify a path is gitignored. But `wrangler.toml` serves the repo root as Cloudflare Pages assets and uses `.assetsignore` as the asset-inclusion filter — these are orthogonal. A path like `bambu-configs/printer-intake-out/` can be git-ignored yet not asset-ignored, passing the guard while remaining deployable.
- **Trigger:** `--out "bambu-configs/printer-intake-out"` → guard passes (gitignored) → candidate skeletons with raw requester notes written → `wrangler pages deploy` serves them publicly.
- **Fix:** Enforce both checks — `git check-ignore -q <path>` AND verify the path is also covered by `.assetsignore` patterns — or simplest/safest: whitelist only `DEFAULT_STAGING` exactly plus paths provably outside the repo root.

---

**HIGH — `assertSafeOutDir` false-rejects the documented safe staging path when the directory does not yet exist**
`scripts/printer-intake-scout.js` — `assertSafeOutDir` function

- **Mechanism:** `.gitignore` lists `scripts/.printer-intake-out/` with a trailing slash (directory-only pattern). `git check-ignore` does not match a non-existent path against a trailing-slash pattern — it returns non-zero, causing the guard to refuse the path. First-time users following the documented default get `exit(2)`.
- **Trigger:** `node scripts/printer-intake-scout.js --queue fixtures/... --out scripts/.printer-intake-out` on a clean clone where the directory has never been created.
- **Practical harm:** Users denied the safe path are likely to try an unsanitised alternative, defeating the entire guard's purpose; the CRITICAL above is the likely outcome.
- **Fix:** Check both `abs` and a synthetic child path (`path.join(abs, 'run-report.json')`) against `git check-ignore`, so directory-type patterns match whether or not the directory exists.

---

### SHOULD-FIX

**MEDIUM — `wrangler kv key list` silently truncates at 1000 entries**
`scripts/printer-intake-scout.js:490–493`

- **Trigger:** Live KV queue exceeds 1000 feedback entries.
- **Fix:** Pass `--limit 100000` if Wrangler supports it for this command, or implement a `--cursor` pagination loop; surface the page ceiling in `report.source`.
